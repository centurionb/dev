Template Description


Pretty is the ultimate and professional way for you to present your portfolio or the very best starting point for your own corporate website. If you are searching something different and out of the line, please check out the Pretty - Clean & Modern Responsive Portfolio Site Template!

Pretty is fully responsive – resize your browser window to see it in action. All the layouts are built with responsive design in mind. It works well on desktops, tablets, and mobile devices. The template is perfectly adapted to various screen sizes, and its attractive fresh design makes it look great.

Pretty is a fully configurable and easy to use HTML website template with a lot of great features. It has been specifically designed to be easily customized and it’s created by using the latest HTML5 and CSS3 techniques. With a responsive design it is easily usable with any device!

Structure Files & Customization

There are several types of files used in the template. There are HTML, CSS, Javascript, Fonts and PHP files. All template files are located in the /Template/ folder. We'll be covering them in details below.

HTML

Here is the list of all html files.

File Name Description
index.html This is the file for your homepage. You can customize it by removing certain parts (blog, testimonials, services,...).

CSS

Here's a list of the stylesheet files we're using with this template, you can find more information opening each file:

File Name Description
mainStyles.css This is the main CSS file and you can use it to modify the template
normalize.css This is css file that's used for HTML5 display definitions
foundation.css This is css file that's used by foundation framework
prettyPhoto.css This is css file that's used by prettyPhoto
jquery.bxslider.css This is css file that's used by bx slider
main.css and mainStyles.css This is the main css file
main.css and mainStyles.css This is the main css file

Javascript

Here's a list of the stylesheet files we're using with this template, you can find more information opening each file:

File Name Description
modernizr.js jQuery file that's used by Modernizr 2.6.1
foundation.js jQuery file that's used to by Foundation Framework
common.js jQuery file that's used for common transitions and animnations
fitVids.js This is the js file used by FitVids 1.0
jquery.prettyPhoto.js jQuery file that's used by PrettyPhoto
jquery.bxslider.js jQuery file that's used by bx slider
jquery.isotope.js jQuery file that's used by isotope plugin
jquery.tweet.js jQuery file that's used to create the twitter feed in the footer

Markup Markup customization and highlights

This template is a responaive layout. All of the information within the main content area is nested within a div with an id of "wrapper". The template has no sidebars.
